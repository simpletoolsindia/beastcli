// Quick Setup

Get started in 30 seconds

No config files needed. Set API keys via environment variables or the --provider flag.

Terminal
# 1. Install npm install -g @simpletoolsindia/beastcli # 2. Set your API key export OPENAI_API_KEY=sk-or-v1-your-key # 3. Run — free model via OpenRouter export OPENAI_BASE_URL=https://openrouter.ai/api/v1 export OPENAI_MODEL=google/gemma-3-4b-it beastcli -p "Hello, write a Python hello world" --bare # That's it. Use --bare for scripting/CI, omit for interactive REPL.
With Ollama (fully local)
# Install ollama from ollama.ai, then: ollama pull qwen2.5-coder:7b export BEASTCLI_PROVIDER=ollama export OPENAI_BASE_URL=http://localhost:11434/v1 export OPENAI_MODEL=qwen2.5-coder:7b beastcli -p "Explain what this function does" --bare

// Usage

CLI modes and examples

BeastCLI runs in interactive REPL mode by default. Use -p for non-interactive output and --bare for minimal scripting/CI mode.

Mode Command Use case
Interactive REPL beastcli Conversational coding sessions
Single prompt beastcli -p "prompt" One-shot tasks, pipes, scripts
Scripting / CI beastcli -p "prompt" --bare Minimal mode — skips UI, auth, plugins
Stdin input cat file.txt | beastcli -p Feed file content as input
Specify provider beastcli --provider ollama Pick provider at startup
Specify model beastcli --model qwen2.5-coder:7b Pick model at startup
// Features

Everything you need

Built on Claude Code's architecture — adapted for open, multi-provider use.

🔌
MCP Server Support
Connect to Model Context Protocol servers. Load configs via --mcp-config.
📁
File Tools
Read, write, edit files with full context. Glob patterns, grep search, and diff view.
⚡
Tool Search
Defer loading built-in tool schemas until needed — saves ~6,000 tokens per request.
🧩
Skills System
Extensible slash commands for /simplify, /loop, /debug, /commit, and more.
🗂️
Task Management
Create, track, and complete tasks. Spawn sub-agents for parallel work.
🔒
Permission Modes
Guidance (ask), AutoPilot (auto-approve), Control (no prompts), Observe (read-only).
🌐
Web Search
Search the web and fetch pages directly from the terminal.
🧠
Context Management
Compact long conversations, auto-detect context limits, session persistence.
// Environment Variables

All configuration options

BeastCLI reads these environment variables for provider and model configuration.

Environment Variables
# Provider selection BEASTCLI_PROVIDER=ollama|openai|openrouter|nvidia # OpenAI-compatible endpoint (all providers except Anthropic) OPENAI_BASE_URL=https://api.openai.com/v1 # API key — picks up automatically from environment OPENAI_API_KEY=sk-... # Model name OPENAI_MODEL=qwen2.5-coder:7b # Anthropic key (for native Claude API) ANTHROPIC_API_KEY=sk-ant-... # NVIDIA NIM key NVIDIA_API_KEY=nvapi-... # Enable tool search (saves ~6K tokens) ENABLE_TOOL_SEARCH=true # Minimal scripting mode (sets BEAST_SIMPLE=1) # Equivalent to passing --bare flag BEAST_SIMPLE=1
